Graphing tool. Make nodes and connect them with edges

  • Press [n] to add a new node.
  • [Click] on a node to select it.
  • Click another node to create an edge between it and the selected node.
  • Press [d] to delete the selected node
  • Press [b] to begin algorithm, choose it from |Algo|
  • Press [r] to reset colours and deselect nodes
  • Click a new location to move the selected node
  • All edges are directed by default, going from white to pink.
  • Create an undirected edge by making an edge both ways
  • Remove an edge by going back over it
  • Note that when adding an undirected edge, you need to manually set them to both have the same edge-weight.
